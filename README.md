# SmartExpense Tracker

A cloud-based expense tracking web application built using Flask and Firebase Firestore.

## Features
- Add expenses
- Delete expenses
- Expense categories
- Pie chart dashboard
- Cloud database integration
- Responsive UI

## Tech Stack
- Python Flask
- Firebase Firestore
- Bootstrap
- Chart.js

## Deployment
- GitHub
- Render
